Overview

Diyalog Messenger Platform(DLG Platfromu) ile kullanıcılarınıza metin, ses, görüntü, video ve dosya gibi birçok içerik türü iletebilirsiniz.

Platformda önceden tanımlanmış bazı mesaj şablonları da vardır. Bu yapılandırılmış mesajları, kullanıcılarınıza iyi bir kullanıcı deneyimi için gönderebilirsiniz.

Desteklenen Mesaj Türleri#
  • Metin
  • Medya (resim, ses, görüntü, videove dosya)
  • Generic Template
  • Button Template
  • Quick Reply Template
  • Tek Seferlik Şifre (OTP-MOK)

Bu mesaj türlerini SendMesssage ve SendOTP apileri aracılığı ile kullanıcılarınıza iletebilirsiniz.

İlettiğiniz mesajların gönderilme, iletilme, okunma gibi sonuçlarını ayrıca müşterilerinizde size iletilen cevapları örneğin seçimli bir mesaj gönderdiğinizde kullanıcınızın hangi seçeneği seçtiği bilgisi gibi durumları DLG Sdk uygulamalarınızı beslemektedir.

DLG platformu, sisteminizdeki her türlü durumu ve olayı besleme yeteneğine sahiptir. Bunu yapmak için, uygulamanızda POST metodunu destekleyen bir http api sağlamalısınız. Platform, http api'nizi otomatik olarak çağırarak durum ve olayları sisteminize besleyecektir.

Detaylı bilgileri webhook dokümanında bulabilirsiniz.

DLG platfrom api'lerini kullanabilmek ve ilettiğiniz mesajların durumlarını alabilmek için yapılası gereken birkaç konfigürasyon işi vardır. Bunları şu şekilde sıralayabiliriz :

  • DLG admin kullanıcısı bir bot kullanıcısı oluşturmalı ve bunun token bilgilerini sizinle paylaşmalıdır. Mesajlaşma API'sini çağırmak için bu token'a ihtiyacınız olacaktır. Aksi takdirde mesaj gönderemezsiniz.
  • Eğer bot üzerinden gönderdiğiniz mesajların durum bilgilerinin uygulamanıza beslenmesini istiyorsanız uygulamanızda POST isteklerini dinleyen bir http api açmalısınız.
  • Mesaj durum bilgilerinin iletilmesi için açtığınız bu http servis adresinin DLG platform yöneticisi tarafından platfrom konfigürasyonuna eklenmesi gerekmektedir.
